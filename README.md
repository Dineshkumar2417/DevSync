# DevSync - Developer Command Center 🚀

A robust, full-stack synchronization tool designed for developers to manage projects, track technical tasks, and secure their digital workflow.

## 🛠️ Tech Stack

- **Frontend:** React.js (Vite)
- **Backend:** Node.js & Express
- **Database:** MongoDB Atlas
- **Authentication:** JWT (JSON Web Tokens) & Bcrypt for secure encryption
- **Deployment:** Render (Backend) & Vercel (Frontend)

## 🌟 Key Features

- **Secure Auth:** Full user authentication system with encrypted password storage.
- **Project Management:** Create, Read, Update, and Delete (CRUD) project data in real-time.
- **Developer Dashboard:** A centralized interface to synchronize all your technical tasks.
- **RESTful API:** Optimized backend routes for seamless data fetching and storage.
- **Clean UI:** Modern, dark-themed interface built for technical efficiency.
